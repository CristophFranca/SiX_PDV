O que tem dentro

Tudo que uma loja precisa,
nada que ela não precisa.

Gestão de Estoque
Cadastro completo com EAN, alertas automáticos por nível (normal, baixo, crítico, zerado), histórico de movimentações e exportação CSV.
PDV Completo
Carrinho, 4 formas de pagamento, cálculo de troco, desconto por venda, abertura e fechamento de caixa com relatório impresso.
Dashboard em Tempo Real
KPIs do dia, gráficos de movimentação dos últimos 14 dias, distribuição por forma de pagamento e lista de produtos para reposição.
Scanner de Código de Barras
Leitura via câmera com BarcodeDetector API. Busca automática por EAN no estoque e adição ao carrinho com um clique ou scan.
Requisições com Impressão
Gere pedidos de reposição formatados para impressão. O débito no estoque é automático ao confirmar a requisição.
Multi-usuário
Perfis Admin e Operador. Admin controla usuários, vê todas as telas e exporta dados. Operador acessa estoque e caixa.
Tecnologia
Stack deliberadamente
simples.

Sem Docker, sem Redis, sem PostgreSQL, sem Node. Python + SQLite + um browser. Se o PC liga, o sistema roda.

Python 3.10+
Flask 3.x
SQLite 3
JavaScript vanilla
Chart.js 4
DM Sans / JetBrains Mono
Instalação
3 minutos do zero
ao primeiro caixa aberto.

Não precisa de servidor, não precisa de técnico. Qualquer pessoa que consiga instalar um programa no Windows consegue instalar o SiX PDV.

1
Instale o Python
Baixe em python.org e marque "Add Python to PATH" durante a instalação.
2
Baixe o SiX PDV
Faça download e extraia para C:\SiXPDV\
3
Clique em iniciar.bat
Instala dependências automaticamente e abre o sistema no navegador.
terminal bash
# Clonar o repositório git clone https://github.com/CristophFranca/SiX_PDV cd six-pdv # Instalar dependências pip install -r requirements.txt # Iniciar python run.py # Acesse http://localhost:5000 # Login: admin@six.local # Senha: admin123
